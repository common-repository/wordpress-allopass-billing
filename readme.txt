Allows easy posting of protected files or entire posts.

== Description ==

Want to monetize the content you create for your blog?

Just simply click on the [allopass button](http://wordpress.org/extend/plugins/wordpress-allopass-billing/screenshots/) and select the document you want to protect.
For post protecting, just [create the document](http://wordpress.org/extend/plugins/wordpress-allopass-billing/screenshots/) and the post will be protected automatically.

== Installation ==

###Updgrading From A Previous Version###

To upgrade from a previous version of this plugin, delete the entire folder and files from the previous version of the plugin and then follow the installation instructions below.

###Installing The Plugin###

Extract all files from the ZIP file, **making sure to keep the file structure intact**, and then upload it to `/wp-content/plugins/`. This should result in multiple subfolders and files.

Then just visit your admin area and activate the plugin.

**See Also:** ["Installing Plugins" article on the WP Codex](http://codex.wordpress.org/Managing_Plugins#Installing_Plugins)


###Plugin Configuration###

To configure this plugin, visit it's settings page. It can be found under the "Allopass" tab in your admin area.

== Frequently Asked Questions ==

= I can't view my documents =

You have to login in the configuration page to view the documents. If you don't have an allopass account, you have to create it through the register form.

= How are my full posts protected? =

You just have to create a document with the url of the post. Then, it will be automatically protected. Note that if you change the permalink settings of your posts, you'll have to create the protected documents again.
